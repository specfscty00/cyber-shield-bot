# Cyber Shield – бот против кибермошенников

**Автор:** Степанчук Максим Вадимович, 12 лет, Брест  
**Конкурс:** #КиберПраво

## Описание
Telegram-бот для обучения распознаванию фишинговых сайтов, мошеннических писем и телефонных звонков.  
- 100 уровней (40 доменов, 35 писем, 25 звонков)  
- Вопросы перемешиваются для каждого пользователя  
- Скам-детектор анализирует ссылки и текст  

## Запуск
1. Установите Python 3.8+  
2. `pip install -r requirements.txt`  
3. Получите токен у @BotFather  
4. Задайте переменную окружения `TOKEN`  
5. `python bot.py`

## Функции
- `/start` – главное меню  
- `/reset` – сброс прогресса  
- Отправьте боту подозрительную ссылку или сообщение – он проанализирует
